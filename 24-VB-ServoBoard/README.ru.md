# VBCores Servo Board v0.2

## Обзор
VBCores Servo Board — компактная плата управления небольшими коллектроными моторами и периферией. На одной плате объединены семь каналов для сстандартных сервоприводов с выделенным питанием, два драйвера коллектоных моторов, интерфейсы A/B энкодеров, GPIO и стандартные коммуникационные интерфейсы.

Плата предназначена для работы в составе мобильных роботов. Удобна для начинающих робототехников для обучения.

Основана на [Микроконтроллерном модуле VBCore32G4](https://github.com/VBCores/VBCores_files/tree/main/01-VB-Core32G4)

### Возможности
- **Модуль микроконтроллера:** VB32G4 на базе STM32G474RE
- **Выходы сервоприводов:** 7 каналов, питание сервоприводов 5.8В, до 5А
- **Драйверы коллекторных двигателей постоянного тока:**
	- 2 × TI DRV8870 драйвера двигателя
	- управление ШИМ от выходов таймеров STM32
	- ограничение тока: 3.3А
- **Варианты питания двигателей:**
	- встроенный выход преобразователя напряжения: 12В или 24В, до 5А
	- питание двигателей от напряжения на разъёме питания
- **Интерфейсы энкодеров:** 2 × интерфейса квадратурных энкодеров
- **Аналоговые/GPIO входы:** 5 каналов, до 3.3В
- **Интерфейсы:**
	- SPI3, логический уровень 3.3В
	- I2C3 с выбираемым питанием разъёма 3.3В / 5В
	- UART, логический уровень 3.3В
	- CAN / CAN FD
- **Измерение входного напряжения:** резистивный делитель 1:16
- **Зуммер**

### Характеристики
- **Входное питание:** 12-45В постоянного тока
- **Питание сервоприводов:** 5.8В, до 5А
- **Питание двигателей:** 12/24В от выхода преобразователя напряжения (до 5А), или напряжение на разъёме питания
- **Ограничение тока двигателя:** 3.3А
- **Микросхема драйвера двигателя:** TI DRV8870, 45В, 3.5А
- **Логический уровень сигнала сервопривода:** 3.3В
- **Логический уровень UART / SPI:** 3.3В
- **Напряжение на входах Analog/GPIO:** до 3.3В
- **Совместимость логических уровней I2C3:** 3.3В и 5В

  
### Размеры
- Плата: 56x101 mm
- Монтажные отверстия: 50x95 mm, Ø2.5 mm

### Motor Driver Pin Mapping

TI DRV8870, 45 V, 3.5 A brushed DC motor driver with PWM control.

| Control       | PIN  | Notes    |
| ------------- | ---- | -------- |
| IN1 (Motor 1) | PB6  | TIM4_CH1 |
| IN2 (Motor 1) | PB4  |  		  |
| IN1 (Motor 2) | PB7  | TIM4_CH2 |
| IN2 (Motor 2) | PB5  | 	   	  |
| VREF          | PB2 | 3.3 A max |
| R_ISEN        | -    | 0.15 Ohm |

### Измерение напряжения на входе платы

Измерение входного напряжения выполняется через резистивный делитель 1:16.

| Управление | PIN | ADC      |
| --------- | --- | -------- |
| VIN_SENSE | PA7 | ADC2_IN4 |

### Перемычки настроек

#### Выбор напряжения питания моторов

| Состояние | Напряжение на моторах |
| ------------ | -------------------- |
| Разомкнут         | 12 V                 |
| Запаян      | 24 V                 |

Эта настройка используется только тогда, когда источник питания моторов выбран как DC-DC. Напряжение питание мотора  не может превышать VIN.

#### Выбор питания для моторов

| Позиция| Источник питания |
| -------- | ------------------- |
| 1        | DC-DC преобразователь: 12В или 24В, 5A макс., в зависимости от запайки перемычки напряжения. |
| 2        | Напряжения с разъема питания VIN, без регулировки |

Motor supply voltage не может превышать VIN.

#### Выбор напряжения для I2C3

I2C3 запаяйте перемычку рядома с разъемом I2C3 тем самым выбирав 3.3В или 5В.

### Примечания
1. Драйверы моторов начинают работать при минимальном напряжении питания 9В.
2. Максимальный ток мотора ограничен значением 3.3A. Настраивается через VREF.
3. Для моторов без энкодеров используйте только клеммы  A и  B (OUT).
4. Напряжение питания сервоприводов составляет 5.8В что близко к максимуму стандарта для радиомоделей. Убедитесь, что ваши серво поддерживают это напряжение.
5. Сигнальные пины серво используют логическое напряжение 3.3В.
6. Максимальное напряжение на разем аналоговых входов составляет 3.3V.  Не подавайте на них напряжение выше.
7. Максимальное напряжение на разъёме SPI3 составляет 3.3В.
8. I2C3 линия совместима с 3.3В и 5В логическими уровнями. Так же вы можете задать напряжение на раъеме 3.3В или 5В запаяв перемычку рядом. Без запаяной перемычки напряжения на ръеме нет. 
9. UART логический уровень составляет 3.3В.
